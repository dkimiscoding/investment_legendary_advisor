# PRD: 미국주식 투자자문봇 (Somong Investment Advisor)

> 버전: 1.0  
> 날짜: 2026-02-12  
> 기반: 소몽의 미국주식 마스터클리스

---

## 1. 개요

### 1.1 제품 비전
**"소몽의 3종 판독기를 AI로 구현한 미국주식 투자자문 서비스"**

개인 투자자가 월가 수준의 투자 판단을 내릴 수 있도록:
- 차트/주가/역발상 3종 판독기 자동 분석
- 실시간 시장 데이터 기반 신호 생성
- 개인화된 매매 타이밍 추천

### 1.2 핵심 가치
| 가치 | 설명 |
|------|------|
| 정확성 | 뉴스 노이즈 제거, 데이터 기반 판단 |
| 실시간 | 24/7 시장 모니터링, 즉시 알림 |
| 개인화 | 투자 성향별 맞춤 전략 |
| 교육 | 투자 원리 이해 + 실전 적용 |

---

## 2. 타겟 사용자

### 2.1 페르소나
- **주린이**: 미국주식 입문자, 체계적 학습 필요
- **직장인**: 시간 부족, 효율적 투자 원함
- **경험자**: 손실 경험, 체계적 판단 도구 필요
- **고수**: 판독기 자동화로 시간 절약

### 2.2 사용자 여정
```
1. 온보딩 → 투자 성향 테스트
2. 학습 → 3종 판독기 원리 이해
3. 실전 → 종목 분석 + 신호 확인
4. 자동화 → 포트폴리오 모니터링
5. 고도화 → 알파 전략 적용
```

## 3. 핵심 기능

### 3.1 3종 판독기 엔진

#### 차트 판독기 (Technical)
| 지표 | 설명 | 점수 |
|------|------|------|
| 이동평균선 | 200일/50일선 상회 여부 | 0~5 |
| 이격도 | 50일선 대비 괴리 정도 | 0~5 |
| RSI | 과매수/과매도 판단 | 0~5 |
| M/W 패턴 | 주/월간 차트 패턴 | 0~5 |

**출력**: 차트점수 (0~20), 공포/중립/광기 판정

#### 주가 판독기 (Valuation)
| 지표 | 설명 | 공식 |
|------|------|------|
| 시장 P/E | S&P500 P/E 대비 | - |
| 종목 P/E | 현재 PER | - |
| 적정주가 | 미래 EPS × 적정PER | EPS(1년후) × 적정PER |
| PEG | 성장성 조정 PER | PER / EPS성장률 |

**출력**: 밸류점수 (0~20), 고평가/적정/저평가 판정

#### 역발상 판독기 (Sentiment)
| 지표 | 설명 | 데이터 소스 |
|------|------|-------------|
| Bull/Bear Spread | 투자자 심리 지표 | AAII |
| Put/Call Ratio | 옵션 심리 | CBOE |
| VIX | 변동성 지수 | CBOE |
| Margin Debt | 신용융자잔고 | FINRA |
| HY Spread | 고수익채 스프레드 | Bloomberg |

**출력**: 센티점수 (0~20), 공포/중립/환호 판정

### 3.2 통합 판독 시스템

#### 3단계 프로세스
```
Step 1: 유동성 판단 (10년 금리, 달러지수, FCI)
    ↓
Step 2: 3종 판독기 실행 (차트+주가+역발상)
    ↓
Step 3: 매매 결정 (매수/매도/관망 + 비중)
```

#### 종합 신호
| 총점 | 판정 | 행동 |
|------|------|------|
| 0-15 | 극도 공포 | 적극 매수 |
| 16-25 | 공포 | 분할 매수 |
| 26-35 | 중립 | 관망/소폭 매수 |
| 36-45 | 환호 | 분할 매도 |
| 46-60 | 극도 환호 | 적극 매도 |

### 3.3 알파 전략 모듈

#### 3.3.1 TOP 20 리스트
- 1군(확신종목): 차트+주가+역발상 모두 우수
- 2군(관심종목): 2개 판독기 우수
- 3군(유망주): 1개 판독기 우수 + 성장성

#### 3.3.2 소외주→주도주 전략
- 하락장에서 낙폭과대주 스크리닝
- 반등 시점 판단 (차트 판독기)
- 모멘텀 확인 후 진입

#### 3.3.3 달리는 말 전략
- YTD 수익률 상위 종목 추적
- 강세장 주도주 선별
- 이격도/RSI로 과열 타이밍 포착

#### 3.3.4 배당 신호등
- 배당률 + 배당성장률 스크리닝
- 배당락일 전후 패턴 분석
- 연금/배당투자자용 장기 포트폴리오

### 3.4 포트폴리오 관리

#### 대시보드
- 총 자산 / 일일 변동 / 누적 수익률
- 섹터별 배분 / 판독기 점수 분포
- 알림 센터 (매수/매도 신호)

#### 리밸런싱 알림
- 월간 리밸런싱 리포트
- 판독기 점수 하락 알림
- 비중 조정 권고

## 10. 부록

### 10.1 핵심 공식 모음
```
[적정주가] = 예상 EPS × 적정 PER
[적정 PER] = EPS 성장률(%) 중앙값
[PEG] = PER / EPS 성장률
[이격도] = (현재가 - 50일선) / 50일선 × 100
```

### 10.2 용어 정의
- **마디선**: 금리 구간별 심리적 저항선 (2%, 3%, 4%, 5%)
- **달리는 말**: 강세장 주도주
- **낙폭과대주**: 하장에서 폭락한 반등 후보 종목
