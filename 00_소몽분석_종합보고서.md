## 1. 투자 철학과 방법론

> [!quote] 소몽의 핵심 철학
> "시장은 유동성이 결정하고, 타이밍은 판독기가 잡고, 수익은 알파 전략이 만든다."

### 1.1 프레임워크 구조

```
┌─────────────────────────────────────────────────────┐
│                  소몽 투자 프레임워크                   │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ① 매크로 환경 판단    ← 유동성 분석 (1단원)          │
│        ↓                                            │
│  ② 시장 기술적 위치    ← 차트 판독기 (2단원)          │
│        ↓                                            │
│  ③ 기업 가치 평가      ← 주가 판독기 (3단원)          │
│        ↓                                            │
│  ④ 심리·센티먼트 확인  ← 역발상 판독기 (4단원)        │
│        ↓                                            │
│  ⑤ 종목 선정·실행      ← 알파 전략 (5단원)           │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 1.2 핵심 원칙

| 원칙 | 설명 |
|------|------|
| **탑다운 접근** | 매크로(금리·유동성) → 섹터 → 개별종목 순서로 분석 |
| **데이터 기반 판단** | 감이 아닌 정량 지표와 기준값으로 판단 |
| **역발상 투자** | 공포에 매수, 탐욕에 경계 — 군중과 반대로 |
| **교차 검증** | 단일 지표가 아닌 복수 판독기의 신호 합산으로 결정 |
| **분산·비중 관리** | 1군/2군/3군 구분, 확신도에 따른 비중 차등 배분 |

## 2. 유동성 분석 (1단원)

> [!tip] 핵심 메시지
> **"돈의 흐름(유동성)이 증시의 방향을 결정한다."**
> 금리·달러·금융상황지수를 삼각 교차검증하여 매크로 환경을 판단한다.

### 2.1 미국 10년물 국채 금리 (강의 1-1)

> [!important] 증시 최대 선행지표
> 미국 10년물 국채 금리는 모든 자산 가격의 기준점(anchor)이다.

**핵심 로직:**

| 금리 방향 | 증시 영향 | 메커니즘 |
|-----------|-----------|----------|
| 금리 ↑ | 증시 ↓ | 할인율 상승 → 미래 현금흐름 현재가치 하락 → 주식 매력도 감소 |
| 금리 ↓ | 증시 ↑ | 할인율 하락 → 미래 현금흐름 현재가치 상승 → 주식 매력도 증가 |

**관찰 포인트:**
- 금리의 **방향**(추세)이 절대 수준보다 중요
- 금리 급등 시 성장주 먼저 타격, 금리 급락 시 성장주 먼저 반등
- 장단기 금리 역전(Yield Curve Inversion) = 경기침체 경고 신호

### 2.2 금융상황지수(NFCI) + 달러지수 교차검증 (강의 1-2)

**이중 필터 시스템:**

```
         NFCI (시카고 연준 금융상황지수)
              ↓
    ┌─────────┴─────────┐
    │                   │
  NFCI ↓ (완화)      NFCI ↑ (긴축)
    │                   │
    ↓                   ↓
  달러 ↓ → ✅ 매수     달러 ↑ → 🚨 매도
  달러 ↑ → ⚠️ 관망     달러 ↓ → ⚠️ 관망
```

| 조합 | NFCI | 달러지수 | 판단 | 행동 |
|------|------|----------|------|------|
| 🟢 강력 매수 | ↓ 완화 | ↓ 약세 | 유동성 풍부 | 공격적 매수 |
| 🟡 관망 A | ↓ 완화 | ↑ 강세 | 혼조 | 선별적 매수 |
| 🟡 관망 B | ↑ 긴축 | ↓ 약세 | 혼조 | 비중 유지 |
| 🔴 강력 매도 | ↑ 긴축 | ↑ 강세 | 유동성 경색 | 비중 축소 |

**데이터 소스:**
- NFCI: [시카고 연준 NFCI](https://www.chicagofed.org/research/data/nfci/current-data)
- 달러지수(DXY): [TradingView DXY](https://www.tradingview.com/symbols/TVC-DXY/)

### 2.3 금리 환경별 유리한 섹터 (강의 1-3)

> [!example] 섹터 로테이션 전략
> 금리 방향에 따라 유리한 섹터가 명확히 구분된다.

| 금리 환경 | 유리한 섹터 | 스타일 | 대표 ETF |
|-----------|-------------|--------|----------|
| **금리 상승기** | 에너지, 금융, 산업재 | 가치주 (Value) | XLE, XLF, XLI |
| **금리 하락기** | 바이오, 리츠, 기술주, 소비재 | 성장주 (Growth) | XBI, VNQ, QQQ |
| **금리 안정기** | 배당주, 유틸리티 | 인컴 (Income) | VYM, XLU |

**로직:** 금리 상승 → 실물경제 활황 → 이익으로 먹는 가치주 유리
금리 하락 → 할인율 감소 → 미래 성장 프리미엄 확대 → 성장주 유리

## 3. 차트 판독기 (2단원)

> [!tip] 핵심 메시지
> **"차트는 시장 참여자들의 합의를 보여주는 거울이다."**
> 이동평균선, 비율, 이격도, RSI, 패턴을 종합하여 5단계 시장 온도를 판정한다.

### 3.1 이동평균선 분석 (강의 2-1)

> [!important] 두 개의 핵심 이동평균선
> - **50일선** = 실적선 (중기 추세)
> - **200일선** = 추세선 (장기 추세)
> - **200주선** = 최후방어선 (초장기 지지)

**판단 기준:**

| 상태 | 조건 | 의미 |
|------|------|------|
| 🟢 **강세** | 주가 > 200일선, 50일선 > 200일선 | 상승 추세 확인 |
| 🟡 **경고** | 주가 < 50일선, 주가 > 200일선 | 단기 조정, 추세 유지 |
| 🟠 **약세 전환** | 50일선이 200일선 하향 돌파 (데스크로스) | 하락 추세 시작 |
| 🔴 **약세** | 주가 < 200일선, 50일선 < 200일선 | 하락 추세 확인 |
| ⚫ **극단적 약세** | 주가 < 200주선 | 최후방어선 붕괴, 대위기 |

**골든크로스 vs 데드크로스:**
- **골든크로스**: 50일선이 200일선을 상향 돌파 → 매수 신호
- **데드크로스**: 50일선이 200일선을 하향 돌파 → 매도 신호
- 200주선 터치 시 = 역사적 매수 기회 (2008, 2020 등)

### 3.2 시장 폭(Market Breadth) 분석 (강의 2-2)

> [!note] S&P500 내 이동평균선 상회 종목 비율
> 시장 전체의 건강 상태를 측정하는 '체온계' 역할

**200일선 상회 종목 비율 기준:**

| 비율 | 상태 | 판단 |
|------|------|------|
| **70% 이상** | 🔴 과열 | 단기 조정 가능성, 신규 매수 자제 |
| **50~70%** | 🟢 건강 | 정상적 상승장, 추세 추종 |
| **30~50%** | 🟡 주의 | 약세 전환 가능성, 리스크 관리 |
| **30% 이하** | 🟢 공포 (역발상 매수) | 과매도 구간, 분할 매수 시작 |

**50일선 상회 종목 비율:** 단기 과열/과매도 판단에 활용
- 80% 이상 = 단기 과열
- 20% 이하 = 단기 과매도 (반등 임박)

**데이터 소스:** [Barchart S&P500 Market Breadth](https://www.barchart.com/stocks/indices/sp-500)

### 3.3 이격도 (강의 2-3)

> [!info] 이격도(Disparity) 정의
> 주가와 이동평균선 사이의 괴리율. 평균으로 회귀하려는 성질을 이용한다.
> **이격도 = (현재주가 - N일 이평선) / N일 이평선 × 100**

**200일선 기준 이격도 판단표:**

| 이격도 | 상태 | 행동 |
|--------|------|------|
| **+20% 이상** | 🔴 극단적 과열 | 적극 차익실현 |
| **+15% ~ +20%** | 🟠 과열 | 분할 매도 시작 |
| **+5% ~ +15%** | 🟢 정상 상승 | 보유 유지 |
| **-5% ~ +5%** | ⚪ 중립 | 이평선 근접, 방향 관찰 |
| **-10% ~ -15%** | 🟢 매수 구간 | 분할 매수 시작 |
| **-15% 이하** | 🟢🟢 강력 매수 | 적극 매수 (역사적 기회) |

### 3.4 RSI 종합 점수 & 5단계 판정 (강의 2-4)

> [!important] 차트 판독기 종합 스코어링
> 이동평균 + 시장폭 + 이격도 + RSI를 합산하여 5단계 판정

**RSI(Relative Strength Index) 기준:**

| RSI | 상태 | 의미 |
|-----|------|------|
| **30 이하** | 과매도 | 매수 신호 |
| **30~50** | 약세 | 관망 또는 소량 매수 |
| **50~70** | 강세 | 추세 추종 |
| **70 이상** | 과매수 | 매도 또는 차익실현 신호 |

**차트 판독기 5단계 종합 판정:**

```
┌──────────────────────────────────────────────────────────┐
│              차트 판독기 5단계 종합 판정                    │
├──────┬──────────┬────────────────────────────────────────┤
│ 단계 │   상태    │              조건                      │
├──────┼──────────┼────────────────────────────────────────┤
│  1   │ 😱 공포  │ RSI<30 + 이격도<-15% + 시장폭<30%      │
│  2   │ 😰 불안  │ RSI 30~40 + 이격도 -10~-15%            │
│  3   │ 😐 중립  │ RSI 40~60 + 이격도 ±5%                 │
│  4   │ 😊 낙관  │ RSI 60~70 + 이격도 +10~+15%            │
│  5   │ 🤪 광기  │ RSI>70 + 이격도>+15% + 시장폭>70%      │
├──────┼──────────┼────────────────────────────────────────┤
│ 행동 │ 1~2=매수 │ 3=관망/리밸런싱 │ 4~5=차익실현/매도     │
└──────┴──────────┴────────────────────────────────────────┘
```

### 3.5 차트 패턴 (강의 2-5)

> [!example] 3대 핵심 패턴

**① M자 패턴 (이중천장, Double Top) — 하락 전환 신호**
```
    ╱╲      ╱╲
   ╱  ╲    ╱  ╲
  ╱    ╲  ╱    ╲
 ╱      ╲╱      ╲
              넥라인 ↓ 이탈 시 매도
```
- 고점 2번 터치 후 하락 → 넥라인 이탈 = 매도 확정
- 하락 목표가 = 넥라인 - (고점 - 넥라인)

**② W자 패턴 (이중바닥, Double Bottom) — 상승 전환 신호**
```
 ╲      ╱╲      ╱
  ╲    ╱  ╲    ╱
   ╲  ╱    ╲  ╱
    ╲╱      ╲╱
              넥라인 ↑ 돌파 시 매수
```
- 저점 2번 터치 후 상승 → 넥라인 돌파 = 매수 확정
- 상승 목표가 = 넥라인 + (넥라인 - 저점)

**③ 컵앤핸들 (Cup & Handle) — 대세 상승 신호**
```
 ╲                ╱─╲
  ╲              ╱   ╲╱ ← 핸들
   ╲            ╱       → 핸들 돌파 시 매수
    ╲    ╱─╲  ╱
     ╲  ╱   ╲╱
      ╲╱
       컵
```
- 대형 U자 형태 후 소규모 조정(핸들) → 핸들 돌파 = 강력 매수
- 장기 상승 추세의 시작점

## 4. 주가 판독기 (3단원)

> [!tip] 핵심 메시지
> **"좋은 기업을 사는 것이 아니라, 좋은 가격에 사는 것이 중요하다."**
> Forward PE, PE밴드, 적정주가, PEG로 기업의 내재가치를 다각도로 평가한다.

### 4.1 Forward PE 분석 (강의 3-1)

> [!info] Forward PE 정의
> Forward PE = 현재 주가 / 향후 12개월 예상 EPS
> 과거가 아닌 **미래 이익** 기준으로 가치를 평가하는 핵심 지표

**S&P500 Forward PE 기준값:**

| Forward PE | 판단 | 역사적 맥락 |
|------------|------|-------------|
| **14배 이하** | 🟢 저평가 | 경기침체·공포 국면 |
| **14~16배** | 🟢 합리적 | 가치 매수 구간 |
| **16~18배** | ⚪ 적정 | 장기 평균 수준 |
| **18~20배** | 🟡 다소 고평가 | 신중한 접근 필요 |
| **20~22배** | 🟠 고평가 | 낙관론 과다 |
| **22배 이상** | 🔴 버블 경고 | 극단적 고평가 |

> [!warning] 주의사항
> - PE는 섹터별로 다르게 적용해야 함 (기술주 평균 PE > 유틸리티 평균 PE)
> - 이익 성장률이 높은 기업은 높은 PE가 정당화될 수 있음 → PEG로 보완

### 4.2 PE 밴드 활용 (강의 3-2)

> [!info] PE 밴드 정의
> 과거 4~5년간 PE의 최고·최저 범위 내에서 현재 주가의 위치를 파악하는 방법

**활용법:**
```
PE 밴드 상단 ─────────── 과거 최고 PE × 현재 EPS ← 고평가 영역
                  │
현재 주가 위치 ←──┤
                  │
PE 밴드 중단 ─────────── 과거 평균 PE × 현재 EPS ← 적정가 영역
                  │
PE 밴드 하단 ─────────── 과거 최저 PE × 현재 EPS ← 저평가 영역
```

| 위치 | 판단 | 행동 |
|------|------|------|
| 밴드 상단 근접/돌파 | 🔴 고평가 | 차익실현 고려 |
| 밴드 중단 부근 | ⚪ 적정 | 보유 유지 |
| 밴드 하단 근접/이탈 | 🟢 저평가 | 매수 기회 |

**데이터 소스:** [Macrotrends PE Ratio](https://www.macrotrends.net/), [GuruFocus](https://www.gurufocus.com/)

### 4.3 적정주가 산출 (강의 3-3)

> [!important] 적정주가 공식
> **적정주가 = 1년 뒤 예상 EPS × 적정 PE**
> **적정 PE = EPS 성장률 × 배수(1~2)**

**단계별 계산법:**

```
Step 1: 1년 뒤 예상 EPS 확인
        → 애널리스트 컨센서스 (Yahoo Finance, Seeking Alpha)

Step 2: EPS 성장률 계산
        → (예상 EPS - 현재 EPS) / 현재 EPS × 100

Step 3: 적정 PE 산출
        → EPS 성장률(%) × 배수
        → 보수적: × 1.0
        → 중립적: × 1.5
        → 공격적: × 2.0

Step 4: 적정주가 = 예상 EPS × 적정 PE

Step 5: 현재가 대비 괴리율 계산
        → (적정주가 - 현재가) / 현재가 × 100
        → +20% 이상 = 매수 매력
        → -20% 이하 = 고평가 경고
```

> [!example] 실전 예시
> - 현재 EPS: $10, 예상 EPS: $13 (성장률 30%)
> - 적정 PE (중립): 30 × 1.5 = 45배
> - 적정주가: $13 × 45 = $585
> - 현재 주가 $500 → 괴리율 +17% → 매수 매력 있음

### 4.4 PEG 분석 (강의 3-4)

> [!info] PEG 정의
> **PEG = PE / EPS 성장률(%)**
> PE만으로 판단할 수 없는 '성장 대비 밸류에이션'을 평가

**PEG 판단 기준:**

| PEG | 판단 | 의미 |
|-----|------|------|
| **< 0.5** | 🟢🟢 극도 저평가 | 시장이 성장을 전혀 반영하지 않음 |
| **0.5 ~ 1.0** | 🟢 강력 매수 | 성장 대비 가격이 매우 합리적 |
| **1.0 ~ 1.5** | ⚪ 적정 | 성장에 부합하는 가격 |
| **1.5 ~ 2.0** | 🟡 다소 고평가 | 프리미엄이 붙기 시작 |
| **> 2.0** | 🔴 고평가 | 성장률 대비 과도한 프리미엄 |

> [!warning] PEG 한계
> - EPS 성장률이 0 이하인 기업에는 적용 불가
> - 성장률 추정치의 정확도에 의존
> - 업종별 평균 PEG가 다름 (기술주 > 유틸리티)

**주가 판독기 종합 체크리스트:**

```
□ Forward PE: S&P500 평균(18배) 대비 위치는?
□ PE 밴드: 과거 4~5년 밴드에서 어디에 위치?
□ 적정주가: 현재가 대비 상승여력은 몇 %?
□ PEG: 성장 대비 가격이 합리적인가?
→ 4개 중 3개 이상 '매수' 판정 시 = 가치 매력 확인
```

## 5. 역발상 판독기 (4단원)

> [!tip] 핵심 메시지
> **"군중이 가장 두려워할 때 사고, 가장 탐욕스러울 때 판다."**
> 5가지 심리·센티먼트 지표를 합산하여 5점 만점 역발상 판독기를 구성한다.

### 5.1 Bull-Bear Spread (강의 4-1)

> [!info] 정의
> AAII 투자자 설문에서 **강세(Bull) 비율 - 약세(Bear) 비율**
> 개인 투자자의 심리 극단을 포착하는 역발상 지표

| Bull-Bear Spread | 심리 상태 | 역발상 판단 |
|------------------|-----------|-------------|
| **-30% 이하** | 😱 극도의 공포 | 🟢🟢 강력 매수 (바닥 근처) |
| **-20% ~ -30%** | 😰 공포 | 🟢 매수 기회 |
| **-20% ~ +20%** | 😐 중립 | ⚪ 관망 |
| **+20% ~ +30%** | 😊 낙관 | 🟡 주의 (고점 접근) |
| **+30% 이상** | 🤩 극도의 탐욕 | 🔴 비중 축소 |

**데이터 소스:** [AAII Sentiment Survey](https://www.aaii.com/sentimentsurvey)

### 5.2 Put/Call Ratio (강의 4-2)

> [!info] 정의
> **Put 거래량 / Call 거래량**
> 옵션 시장에서 헤지(풋) vs 투기(콜) 비율로 공포/탐욕 측정

| Put/Call Ratio | 심리 상태 | 역발상 판단 |
|----------------|-----------|-------------|
| **1.2 이상** | 😱 극도의 공포 | 🟢🟢 강력 매수 |
| **1.0 ~ 1.2** | 😰 공포 | 🟢 매수 기회 |
| **0.7 ~ 1.0** | 😐 중립 | ⚪ 관망 |
| **0.5 ~ 0.7** | 😊 낙관 | 🟡 주의 |
| **0.5 이하** | 🤩 극도의 탐욕 | 🔴 비중 축소 |

**데이터 소스:** [CBOE Put/Call Ratio](https://www.cboe.com/us/options/market_statistics/)

### 5.3 Margin Debt (강의 4-3)

> [!info] 정의
> 투자자 신용 잔고(빚투)의 전년 대비 변화율(YoY)
> 레버리지 수준으로 시장의 과열/침체를 판단

| Margin Debt YoY | 상태 | 역발상 판단 |
|-----------------|------|-------------|
| **+40% 이상 후 꺾임** | 📈→📉 고점 형성 | 🔴 매도 신호 (레버리지 정점 후 디레버리징) |
| **+20% ~ +40%** | 📈 가속 상승 | 🟡 경계 (과열 접근) |
| **0% ~ +20%** | 📈 완만 상승 | ⚪ 정상 |
| **-20% ~ 0%** | 📉 감소 | 🟡 조정 국면 |
| **-20% 이하 후 반등** | 📉→📈 저점 형성 | 🟢 매수 신호 (디레버리징 종료) |

**핵심 포인트:** 절대 수준보다 **변화의 방향 전환점(꺾임)**이 중요

**데이터 소스:** [FINRA Margin Statistics](https://www.finra.org/investors/learn-to-invest/advanced-investing/margin-statistics)

### 5.4 High Yield Spread (강의 4-4)

> [!info] 정의
> **하이일드 채권 수익률 - 국채 수익률**
> 신용 리스크 프리미엄으로 시장의 진짜 위기 vs 가짜 위기를 구별

| HY Spread | 상태 | 판단 |
|-----------|------|------|
| **7% 이상** | 🔴 금융위기 수준 | 진짜 위기 — 현금 확보 우선, 이후 매수 |
| **5% ~ 7%** | 🟠 심각한 스트레스 | 진짜 위기 가능성 높음 — 경계 |
| **5% 이하 & 안정** | 🟢 가짜 위기 | **주가 하락 = 매수 기회!** |
| **3% ~ 4%** | ⚪ 정상 | 신용시장 안정 |
| **3% 이하** | 🟡 과도한 안도감 | 리스크 과소평가 경계 |

> [!warning] 핵심 판별법
> 주식시장이 급락했는데 **HY Spread가 5% 이하로 안정적**이면 = **가짜 위기** = 매수 기회
> 주식시장이 급락하고 **HY Spread가 5% 이상 급등**하면 = **진짜 위기** = 현금 확보

**데이터 소스:** [FRED ICE BofA HY Spread](https://fred.stlouisfed.org/series/BAMLH0A0HYM2)

### 5.5 VIX + 종합 판독기 (강의 4-5)

> [!info] VIX (공포지수) 정의
> S&P500 옵션의 내재 변동성. 시장의 단기 공포 수준을 직접 측정

| VIX | 상태 | 역발상 판단 |
|-----|------|-------------|
| **40 이상** | 😱 극단적 공포 | 🟢🟢 강력 매수 |
| **30 ~ 40** | 😰 공포 | 🟢 매수 기회 |
| **20 ~ 30** | 😐 불안 | ⚪ 관망 |
| **15 ~ 20** | 😊 안정 | 정상 시장 |
| **12 이하** | 😴 안일 | 🟡 경계 (급변 가능) |

**데이터 소스:** [CBOE VIX](https://www.cboe.com/tradable_products/vix/)

---

### 역발상 판독기 종합 스코어 (5점 만점)

> [!important] 5가지 지표를 합산하여 종합 역발상 점수를 산출

| 지표 | 매수 신호 (1점) | 중립 (0점) | 매도 신호 (-1점) |
|------|-----------------|------------|------------------|
| **Bull-Bear Spread** | < -20% | -20% ~ +20% | > +20% |
| **Put/Call Ratio** | > 1.0 | 0.7 ~ 1.0 | < 0.5 |
| **Margin Debt YoY** | -20%↓ 후 반등 | 0~+20% | +40%↑ 후 꺾임 |
| **HY Spread** | < 5% & 안정 (가짜위기) | 3~4% 정상 | > 5% (진짜위기) |
| **VIX** | > 30 | 15~30 | < 12 |

**종합 판정:**

| 합산 점수 | 판정 | 행동 |
|-----------|------|------|
| **+4 ~ +5점** | 🟢🟢 극단적 공포 | 최대 비중 매수 (역사적 기회) |
| **+2 ~ +3점** | 🟢 공포 | 분할 매수 시작 |
| **-1 ~ +1점** | ⚪ 중립 | 관망, 기존 전략 유지 |
| **-2 ~ -3점** | 🟡 탐욕 | 비중 축소, 차익실현 |
| **-4 ~ -5점** | 🔴 극단적 탐욕 | 방어적 포지션 전환 |

## 6. 알파 전략 (5단원)

> [!tip] 핵심 메시지
> **"판독기로 시장의 온도를 재고, 알파 전략으로 종목을 고른다."**
> TOP20 거름망, 소외주, 주도주 — 3가지 전략으로 시장 상황별 최적 종목을 선정한다.

### 6.1 TOP20 거름망 전략 (강의 5-1)

> [!important] 비중 관리의 핵심
> 모든 종목을 동일 비중으로 담지 않는다. 확신도에 따라 1군/2군/3군으로 분류하고 비중을 차등 배분한다.

**종목 분류 기준:**

| 등급 | 비중 | 조건 | 특징 |
|------|------|------|------|
| **1군** (핵심) | **60%** | 판독기 3종 모두 매수 + 강한 펀더멘털 | 대형 우량주, 확신도 최상 |
| **2군** (보조) | **30%** | 판독기 2종 매수 + 양호한 펀더멘털 | 성장주, 섹터 대표주 |
| **3군** (탐색) | **10%** | 판독기 1종 매수 + 특수 상황 | 소외주, 턴어라운드 후보 |

**거름망 프로세스:**

```
전체 미국주식 (~6,000종)
        ↓ 1차 필터: 시가총액 $10B 이상
     ~500종
        ↓ 2차 필터: 차트 판독기 통과 (200일선 위, RSI 적정)
     ~200종
        ↓ 3차 필터: 주가 판독기 통과 (PEG < 2, Forward PE 적정)
     ~50종
        ↓ 4차 필터: 역발상 판독기 교차검증
     ~20종 (TOP20)
        ↓ 5차: 1군/2군/3군 분류 & 비중 배분
     최종 포트폴리오
```

### 6.2 소외주 전략 (강의 5-2)

> [!note] "시장이 잊은 보석을 찾아라"
> 일시적 악재로 과매도된 우량주를 역발상으로 매수하는 전략

**소외주 선정 조건 (AND 조건):**

| 조건 | 기준 | 근거 |
|------|------|------|
| **RSI** | 30 이하 | 기술적 과매도 확인 |
| **배당률** | 역사적 고배당률 (상위 20% 수준) | 주가 하락으로 배당률 상승 = 저평가 시그널 |
| **펀더멘털** | 이익 체력 유지 (적자 전환 아님) | 일시적 악재 vs 구조적 문제 구별 |
| **HY Spread** | 5% 이하 (가짜 위기) | 시스템 리스크가 아닌 개별 악재 확인 |

**소외주 매수 타이밍:**
1. RSI 30 이하 진입 → 1차 분할매수 (30%)
2. RSI 20 이하 추가 하락 → 2차 분할매수 (30%)
3. RSI 30 상향 돌파 → 3차 확인 매수 (40%)

> [!example] 소외주 전략 적합 사례
> - 대형 우량주가 일시적 실적 미스로 급락
> - 섹터 전체 하락 시 가장 펀더멘털이 강한 기업
> - 시장 공포 국면에서 배당률이 역사적 상단에 도달한 종목

### 6.3 주도주 전략 (강의 5-3)

> [!note] "강한 말에 올라타라"
> 상승 모멘텀이 강한 종목에 추세 추종하는 전략

**주도주 선정 조건 (AND 조건):**

| 조건 | 기준 | 근거 |
|------|------|------|
| **RSI** | 50~70 | 강세이면서 아직 과매수 아닌 구간 |
| **신고가** | 52주 신고가 근접 또는 돌파 | 강한 수요 확인 |
| **상대강도** | 섹터/시장 대비 상대강도 상위 | 시장보다 강한 종목 |
| **200일선** | 200일선 위에서 상승 추세 | 장기 추세 확인 |

**주도주 매수 타이밍:**
1. 52주 신고가 돌파 + 거래량 증가 → 돌파 매수
2. 신고가 돌파 후 눌림목(RSI 50 부근) → 풀백 매수
3. 컵앤핸들 패턴 완성 후 핸들 돌파 → 패턴 매수

> [!warning] 주도주 전략 주의사항
> - RSI 70 이상 과매수 구간 진입 시 추격 매수 금지
> - 200일선 이탈 시 즉시 손절
> - 시장 전체가 약세(역발상 판독기 -3 이하)일 때는 주도주 전략 중단

### 6.4 전략 선택 매트릭스

| 시장 상황 | 역발상 판독기 | 추천 전략 |
|-----------|---------------|-----------|
| 극단적 공포 | +4 ~ +5 | **소외주** 집중 (1군 비중 확대) |
| 공포 | +2 ~ +3 | 소외주 + TOP20 1군 매수 |
| 중립 | -1 ~ +1 | TOP20 리밸런싱, 관망 |
| 낙관 | -2 ~ -3 | **주도주** 추세 추종 (비중 축소 시작) |
| 극단적 탐욕 | -4 ~ -5 | 차익실현, 현금 비중 확대 |
